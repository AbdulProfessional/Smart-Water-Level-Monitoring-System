# Smart Water Level Monitoring System

## Overview

The Smart Water Level Monitoring System is an IoT-based solution designed to monitor water levels in tanks and prevent overflow. The system uses an ultrasonic sensor to measure the water level, displays the level on an OLED screen, and sends notifications to users through the Blynk application.

## Features

* Real-time water level monitoring
* OLED display for local monitoring
* Blynk-based remote monitoring
* High water level alert notifications
* Overflow prevention
* Low-cost and easy-to-use system

## Components Used

* NodeMCU ESP8266
* HC-SR04 Ultrasonic Sensor
* OLED Display (SSD1306)
* Jumper Wires
* USB Power Supply
* Blynk IoT Platform

## Applications

* Homes and Apartments
* Hostels
* Schools and Colleges
* Offices
* Small Industries
